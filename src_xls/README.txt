本zipファイルには、「＜解析塾秘伝＞有限要素法の作り方!」で紹介されている例題を、
Microsoft Excel VBAで実行するためのプログラムファイルが収められています。
ファイル名はテキストの章・節・項の番号に対応しています。
本プログラムは、Excel 2010で動作確認しています。
